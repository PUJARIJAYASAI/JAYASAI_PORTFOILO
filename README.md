# Jayasai Pujari | Software Engineer & AI/ML Developer

A high-performance, engineering-first portfolio featuring a dark, terminal-inspired aesthetic. Built with a focus on performance, accessibility, and clean system architecture.

## 🚀 Key Features

- **Professional Aesthetic**: Dark mode "Cyber-Engineering" design with high-trust terminal green accents.
- **Performance Optimized**: Lighthouse-optimized Largest Contentful Paint (LCP) and efficient asset loading.
- **Responsive Architecture**: Fully functional across desktop, tablet, and mobile devices.
- **Project Showcase**: Detailed case studies of AI/ML systems (IDS, Cop Connect, Health Intake).
- **Interactive Elements**: Smooth scrolling, intersection-based navigation, and staggered reveal animations.

## 🛠️ Technical Stack

- **Core**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Build Tool**: Vite
- **Assets**: Custom technical mockups and vector icons

## 📦 Local Development

To run the project locally, ensure you have [Node.js](https://nodejs.org/) installed:

1. Clone the repository:
   ```bash
   git clone https://github.com/PUJARIJAYASAI/JAYASAI_PORTFOILO.git
   ```
2. Navigate to the project directory:
   ```bash
   cd JAYASAI_PORTFOILO
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
